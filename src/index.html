Control is done with the mouse.

  1. Aim the cursor at the spot where you want to launch the ball
  2. To strike the ball, click and hold the left mouse button, then move it in the direction of the cue, just as if you were pulling back the cue for a shot in real life.
  3. The further you pull back the cue, the stronger the shot will be!
  4. To make the shot, release the left mouse button
  5. To select a ball for a shot, quickly click on it with the left mouse button. You cannot select a ball if it's the first shot or if the balls are still in motion
  6. To start over, click the button with an circle arrow icon in the upper right corner of the screen

Enjoy the game!
